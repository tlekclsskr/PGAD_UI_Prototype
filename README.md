# PGAD (Personal Gait Analysis Dashboard)

A mobile application for tracking and analyzing user's walking patterns with AI-powered anomaly detection system for gait behavior monitoring.

## Key Features

- **Gait Analysis**: Track daily walking distance and gait behavior patterns
- **Anomaly Detection**: AI-powered system to detect abnormal walking patterns
- **Data Dashboard**: Real-time walking statistics and insights
- **In-depth Analytics**: Daily, monthly, and yearly data analysis with interactive charts
- **Personal Profile**: User data management and app settings
